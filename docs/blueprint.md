# **App Name**: EduCog Micro

## Core Features:

- Digitized Chapter Viewer: Provides a user-friendly interface for viewing digitized course materials.
- Full-Text Search: Enables fuzzy matching search across all digitized course chapters.
- Interactive Knowledge Graph: Presents interactive visualizations of knowledge domains and relationships in each lesson.
- AI Study Assistant: Offers AI-powered assistance to students by answering questions based on course materials and suggesting study strategies, incorporating page numbers into the response using a tool.
- Learning Analytics Dashboard: Allows instructors to gain insight into student understanding.
- Personalized Content Recommendations: Provides personalized learning material recommendations based on weak areas from quiz scores.
- WebAssembly Simulation Interface: Enable manual specification of parameters like frequency and duty cycle in a WASM environment.

## Style Guidelines:

- Primary color: Light desaturated blue (#A7D1FF), evoking the feeling of technical expertise.
- Background color: Dark blue (#0A2647), a serious and calm base for prolonged study.
- Accent color: Electric purple (#8900FF) for highlights and important interactive elements.
- Body and headline font: 'Inter', a sans-serif, provides a modern and objective aesthetic.
- Code font: 'Source Code Pro', a monospaced font, will be used for displaying code snippets within the chapter content and simulation environments.
- Use a set of minimalist icons to represent different types of content, such as videos, quizzes, and simulations.
- Employs a split-screen layout with a navigation sidebar for chapters and a main content area for reading and interacting with the material.
