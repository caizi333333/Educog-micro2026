# Google AI API 设置指南

## 概述
"芯智育才"平台使用 Google Gemini AI 来提供智能化的学习体验：
- **AI学习助教**：智能回答8051微控制器相关问题
- **个性化学习计划**：根据测评结果生成定制学习路径
- **智能仿真分析**：深度理解汇编代码逻辑和故障影响

如果没有配置有效的API密钥，系统会自动使用内置的备用功能。

## 解决方案

### 选项1：使用内置备用功能（推荐）
**无需任何配置**，系统已经实现了完整的备用功能：
- ✅ 自动检测API密钥错误
- ✅ 立即切换到内置功能
- ✅ AI助教：提供8051相关知识的详细回答
- ✅ 学习计划：基于知识点分析的完整学习路径
- ✅ 仿真分析：精确的8051汇编代码执行模拟

### 选项2：配置Google AI API密钥

如果您希望使用完整的AI功能，请按以下步骤操作：

#### 1. 获取Google AI API密钥
1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
2. 使用Google账号登录
3. 点击"Create API key"创建新的API密钥
4. 复制生成的API密钥（以"AIza"开头的字符串）

#### 2. 配置环境变量
编辑项目根目录的 `.env.local` 文件：

```bash
# 原来的配置
GOOGLE_GENAI_API_KEY=your_actual_google_ai_api_key_here

# 改为你的真实API密钥
GOOGLE_GENAI_API_KEY=AIzaxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

#### 3. 重启开发服务器
```bash
npm run dev
```

#### 4. 验证配置
- 访问AI助教页面，提问关于8051的问题
- 访问个性教学页面，生成学习计划
- 访问实验仿真页面，运行代码分析

## 故障排除

### 常见错误
1. **"API key not valid"**：API密钥无效或过期
2. **网络超时**：检查网络连接
3. **配额用尽**：等待下月配额重置或升级付费计划

### 解决方法
所有错误都会自动fallback到内置功能，确保平台正常使用。
