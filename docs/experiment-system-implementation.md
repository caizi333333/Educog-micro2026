# 8051微控制器实验系统重构与实现

## 概述

本次实现对桂林航天工业学院的8051微控制器仿真教育平台进行了全面的重构和增强，实现了专业级的实验管理系统，包含8个完整的8051微控制器实验课程。

## 核心特性

### 1. 专业实验管理中心
- **分类筛选**: 按难度（基础/中级/高级）快速筛选实验
- **智能排序**: 实验按编号自动排序，显示预计用时
- **分类浏览**: 按实验类别（IO端口、定时器、显示控制等）浏览
- **一键加载**: 点击即可加载完整实验代码和配置

### 2. 丰富的实验内容
实现了8个标准8051微控制器实验：

#### 实验一：基础指令与寄存器操作
- **类别**: 基础编程 | **难度**: 基础 | **用时**: 45分钟
- **目标**: MOV指令、寄存器操作、算术运算、逻辑运算
- **知识点**: 基础指令集、寄存器结构、数据传送、算术逻辑运算

#### 实验二：P1口LED流水灯控制
- **类别**: IO端口应用 | **难度**: 基础 | **用时**: 60分钟
- **目标**: P1口输出控制、位操作、多种LED显示模式
- **特色**: 三种模式 - 单点流水、奇偶交替、中心扩散

#### 实验四：数码管显示实验
- **类别**: 显示控制 | **难度**: 中级 | **用时**: 90分钟
- **目标**: 7段数码管驱动、动态扫描、BCD译码
- **特色**: 4位数码管计数器，支持0000-9999循环计数

#### 实验五：按键输入与消抖处理
- **类别**: 输入处理 | **难度**: 中级 | **用时**: 75分钟
- **目标**: 矩阵键盘扫描、软件消抖、中断处理
- **特色**: 4x4矩阵键盘，支持中断和查询两种处理方式

#### 实验六：定时器中断与计时功能
- **类别**: 定时器应用 | **难度**: 中级 | **用时**: 100分钟
- **目标**: 实时时钟、定时器中断、精确时间计算
- **特色**: 数字时钟显示，可按键设置时间，支持时分/分秒切换

#### 实验七：蜂鸣器音频控制
- **类别**: 音频控制 | **难度**: 中级 | **用时**: 80分钟
- **目标**: 蜂鸣器驱动、音频频率控制、音乐播放
- **特色**: 播放《小星星》音乐，支持音符编码和节拍控制

#### 实验八：步进电机控制实验
- **类别**: 电机控制 | **难度**: 高级 | **用时**: 110分钟
- **目标**: 步进电机驱动、相序控制、精确定位
- **特色**: 四相八拍控制，支持正反转、调速、步进定位

### 3. 增强的实验详情界面
- **标签式布局**: 目标、知识点、硬件、结果四个标签
- **知识点标签**: 核心概念以标签形式展示
- **硬件需求**: 详细的硬件连接要求
- **预期结果**: 清晰的实验成功标准
- **故障排除**: 常见问题及解决方案
- **扩展实验**: 进阶学习建议

### 4. 完善的代码结构
每个实验包含：
- **完整汇编代码**: 从头到尾可运行的8051汇编程序
- **详细注释**: 中英文对照，解释每条指令的作用
- **模块化设计**: 主程序、子程序、中断服务程序分离
- **标准格式**: 遵循8051汇编编程规范

## 代码示例

### 实验二：P1口LED流水灯控制
```assembly
; 桂林航天工业学院 - 实验二：P1口LED流水灯控制
; 功能: 8个LED实现多种流水灯模式，奇偶交替闪烁
ORG 0000H
LJMP MAIN

MAIN:
MODE1:  ; 模式1: 单点流水灯
    MOV P1, #0FEH    ; 11111110B - P1.0亮
    ACALL DELAY
    MOV P1, #0FDH    ; 11111101B - P1.1亮  
    ACALL DELAY
    ; ... 其他7个LED依次点亮
MODE2:  ; 模式2: 奇偶交替闪烁
    MOV P1, #0AAH    ; 10101010B - 奇数位LED亮
    ACALL DELAY
    MOV P1, #55H     ; 01010101B - 偶数位LED亮
    ACALL DELAY
```

### 实验六：定时器中断时钟
```assembly
; 实时时钟系统，支持按键设置时间
ORG 0000H
LJMP MAIN

ORG 000BH        ; T0中断向量
LJMP T0_INT

MAIN:
    ; 定时器T0初始化 - 用于1秒计时
    MOV TMOD, #11H   ; T0和T1都是模式1
    MOV TH0, #3CH    ; 50ms定时
    MOV TL0, #0B0H
    SETB ET0         ; 允许T0中断
    SETB EA          ; 开总中断
    SETB TR0         ; 启动T0
```
