## 详细实验指导

### 实验一：指令系统实验

**实验目的**
1. 掌握8051基本指令的使用方法
2. 理解立即寻址、直接寻址、寄存器寻址等寻址方式
3. 学会使用循环结构和条件跳转指令
4. 掌握子程序调用和返回机制

**知识点分析**
- **MOV指令**: 数据传送指令，支持多种寻址方式
- **RL/RR指令**: 循环移位指令，用于LED流水效果
- **CJNE指令**: 比较跳转指令，用于循环控制
- **ACALL/RET**: 子程序调用和返回
- **PUSH/POP**: 堆栈操作，保护和恢复现场

**硬件连接**
```
P1.0 ——— LED0 ——— 330Ω ——— VCC
P1.1 ——— LED1 ——— 330Ω ——— VCC
...
P1.7 ——— LED7 ——— 330Ω ——— VCC
```

**代码详解**
```assembly
; 立即寻址示例
MOV P1, #0FEH    ; 将立即数0FEH送入P1口
                 ; #表示立即寻址，0FEH为十六进制数

; 循环移位示例
RL A             ; 累加器A循环左移
                 ; 11111110B -> 11111101B

; 比较跳转示例
CJNE A, #7FH, LOOP1  ; A与7FH比较，不等则跳转到LOOP1
```

**实验步骤**
1. 打开"芯智育才"仿真平台
2. 选择"实验一：指令系统实验"
3. 观察代码结构和注释
4. 点击"运行仿真"
5. 观察LED流水灯效果
6. 尝试修改延时参数，观察速度变化

**预期结果**
- LED从右向左依次点亮（P1.0→P1.7）
- 到达左端后从左向右依次点亮（P1.7→P1.0）
- 整个过程循环进行，速度均匀

**故障排除**
| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| LED不亮 | 初始化错误 | 检查P1口初始化值 |
| 流水过快 | 延时不足 | 增加DELAY子程序计数值 |
| 方向错误 | 移位指令错误 | 检查RL/RR指令使用 |

### 实验三：定时/计数器实验（重点实验）

**实验目的**
1. 掌握8051定时器/计数器的工作原理
2. 理解TMOD寄存器的配置方法
3. 学会计算定时器初值
4. 掌握中断系统的使用

**理论基础**

**1. 定时器工作原理**
8051有两个16位定时器T0和T1，可以工作在4种模式：
- 模式0：13位定时器/计数器
- 模式1：16位定时器/计数器（常用）
- 模式2：8位自动重装定时器/计数器
- 模式3：T0分成两个8位定时器

**2. TMOD寄存器配置**
```
TMOD = GATE C/T M1 M0 GATE C/T M1 M0
       |----T1----|  |----T0----|

GATE: 门控位，0=定时器启动不受外部中断影响
C/T:  定时/计数选择，0=定时器模式，1=计数器模式
M1M0: 工作模式选择
      00=模式0, 01=模式1, 10=模式2, 11=模式3
```

**3. 定时器初值计算**
```
机器周期 = 12 / 晶振频率 = 12 / 12MHz = 1μs
定时时间 = 计数值 × 机器周期
计数值 = 定时时间 / 机器周期

例：定时50ms
计数值 = 50ms / 1μs = 50000
初值 = 65536 - 50000 = 15536 = 3CB0H
因此：TH0 = 3CH, TL0 = B0H
```

**4. 中断向量表**
```
中断源     中断向量地址
复位       0000H
外部中断0  0003H
定时器T0   000BH
外部中断1  0013H
定时器T1   001BH
串口       0023H
```

**代码分析**
```assembly
; 定时器配置
MOV TMOD, #01H   ; T0模式1，16位定时器
                 ; TMOD = 00000001B

; 初值设置
MOV TH0, #3CH    ; 高8位初值
MOV TL0, #0B0H   ; 低8位初值

; 中断设置
SETB ET0         ; 允许T0中断 (IE.1=1)
SETB EA          ; 总中断允许 (IE.7=1)
SETB TR0         ; 启动T0 (TCON.4=1)

; 中断服务程序
T0_INT:
    MOV TH0, #3CH    ; 重装初值（模式1需要）
    MOV TL0, #0B0H
    DJNZ R0, EXIT    ; 软件分频
    MOV R0, #20      ; 重装计数值
    CPL P0.0         ; 翻转输出
EXIT:
    RETI             ; 中断返回
```

**实验扩展**
1. **改变方波频率**：修改R0初值
   - R0=10：方波周期=1秒
   - R0=40：方波周期=4秒

2. **双定时器应用**：同时使用T0和T1
```assembly
; T1配置为模式2，产生波特率
MOV TMOD, #21H   ; T1模式2，T0模式1
```

3. **精确计时**：考虑中断响应时间
```assembly
; 补偿中断响应延迟
MOV TH0, #3DH    ; 略微调整初值
MOV TL0, #00H
```

### 实验九：串口通信实验（新增）

**实验目的**
1. 掌握8051串口通信原理
2. 学会配置串口工作模式和波特率
3. 实现字符收发和简单协议

**理论基础**

**1. 串口工作模式**
8051串口有4种工作模式：
- 模式0：同步移位寄存器
- 模式1：8位UART，波特率可变（常用）
- 模式2：9位UART，波特率固定
- 模式3：9位UART，波特率可变

**2. SCON寄存器配置**
```
SCON = SM0 SM1 SM2 REN TB8 RB8 TI RI

SM0SM1: 工作模式选择
        00=模式0, 01=模式1, 10=模式2, 11=模式3
REN:    接收允许位，1=允许接收
TI:     发送中断标志位
RI:     接收中断标志位
```

**3. 波特率计算**
模式1波特率计算公式：
```
波特率 = (2^SMOD/32) × (晶振频率/(12×[256-TH1]))

对于9600波特率，12MHz晶振：
9600 = (1/32) × (12000000/(12×[256-TH1]))
解得：TH1 = 253 = FDH
```

**实验步骤**
1. 配置串口为模式1
2. 设置波特率为9600
3. 允许串口中断
4. 发送欢迎消息
5. 实现字符回显功能

**应用扩展**
1. **命令解析器**：实现简单的AT命令
2. **数据包传输**：添加帧头、校验和帧尾
3. **多机通信**：实现主从机通信协议

### 实验十：ADC模数转换实验（新增）

**实验目的**
1. 掌握ADC工作原理和接口技术
2. 学会编写ADC控制程序
3. 实现电压测量和显示功能

**ADC0809技术参数**
- 分辨率：8位（256个量化级别）
- 转换时间：约100μs
- 输入电压范围：0-5V
- 转换精度：±1LSB

**电压计算公式**
```
输入电压 = (ADC值 / 255) × 参考电压

例：ADC值=128，参考电压=5V
输入电压 = (128/255) × 5V ≈ 2.51V
```

**控制时序**
1. 输出通道地址到A0-A2
2. 发送ALE锁存地址
3. 发送START启动转换
4. 等待EOC变高（转换完成）
5. 发送OE读取数据

## 常见问题与解答

**Q1: 为什么LED不亮？**
A1: 检查LED连接极性，8051的I/O口低电平时LED亮。

**Q2: 定时不准确怎么办？**
A2: 检查晶振频率和初值计算，考虑中断响应时间。

**Q3: 串口通信乱码如何解决？**
A3: 检查波特率设置，确保发送和接收端波特率一致。

**Q4: ADC转换值不稳定？**
A4: 添加软件滤波，多次采样取平均值。
