## 项目恢复指南

要从备份恢复完整项目：

1. **复制备份文件夹**到目标位置
2. **安装依赖**: `npm install`
3. **配置环境**: 复制`.env.example`为`.env`并填入配置
4. **初始化数据库**: `npm run prisma:push`
5. **启动开发服务器**: `npm run dev`
